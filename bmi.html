During the London App Brewery's express tutorials I learned how to process get requests. This project is a BMI calculator. Initially this project was an introduction project to learn about handling get requests from a form. After I completed the EJS and express tutorials I came back to this project to practice handling post requests as well as using partials and templating. To create a complete application I implemented a simle design and then built upon the results page with some conditional logic to output the users weight range results.
